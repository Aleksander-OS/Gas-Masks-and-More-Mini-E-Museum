Afrika Korps MG34 ammunition chest

Patr Kast 41f MG or more precisely Patronenkasten 41 f√ºr Maschinengewehr which means "41 ammo box for a machine gun" is a German improved ammunition box for the MG34 which, unlike the older Patronenkasten 34 model, used a different mounting that was cheaper and less breakable, which was characteristic of the older model, it also had a seal, which made the box waterproof. These boxes held 300 7.92mm rounds on a belt. This corner one was used by the Afrika Korps, as evidenced by the Dunkelgelb color, which was hidden under a layer of non-original paint for a very long time. Unfortunately, I couldn't find the manufacturer's code for this box and it probably isn't on it.

This is the only item used by Afrika Korps in my collection at the time of writing (22/02/2024)
